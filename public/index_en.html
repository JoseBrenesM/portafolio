Here are some of my most recent projects, along with a brief description of each one.

Veterinaria El Ganadero

A full web application built as my final university project for a real company. Developed with .NET using a layered architecture, with a SQL database deployed on Azure, along with Azure App Service hosting.

Features include JWT-based authentication, role-based authorization, pet registration, appointment scheduling, and a point-of-sale module. Payments are integrated through the PayPal API to support card transactions.

This project was developed entirely by me as a FullStack developer.

Casa de la Hamaca

An informational website for a small business that showcases available products and includes an admin area for product management. Built by me as a FullStack developer using .NET, Azure, SQL, and Blob Storage for image management.

Facebook Clone

A Facebook-inspired web app built with Node.js and Express for the backend, using JWT for authentication and bcrypt for password encryption. Data is stored in MongoDB.

Includes posts, creating publications, likes, comments, replies, a marketplace section, and user profile editing—handled through APIs that process user requests.
