Welcome to My Portfolio

Here you will find some of my most recent projects along with their explanations.

Veterinaria El Ganadero

Website developed as a final university project, which involves creating a web application for a real company. This web is built with .NET using a layered architecture. The database is developed in SQL and deployed on Azure, as is the web host. This website includes many functionalities such as authentication via JSON Web Tokens, role-based authorization, allowing clients to register pets, schedule appointments, and its main feature, a point-of-sale system that uses the PayPal API for credit card payments. This project was entirely developed by me as a FullStack developer.

Casa de la Hamaca

Informational website for a small business that showcases available products, along with an administrative section for product management. Developed by myself as a Full Stack developer using .NET, Azure, SQL, and Blob Storage for image handling.

Facebook Clone

Website that simulates Facebook, using Node.js along with Express for the backend. It includes functionalities such as jsonwebtoken for user authentication, bcrypt for password encryption, and stores everything in MongoDB databases. It features a posts section, creating posts, likes, comments, replies, as well as a commerce section and user profile editing, all managed through APIs that handle user requests.
